# Retail-Sales_and_Feedback_Dashboard
# 📊 Retail Sales & Customer Feedback Dashboard

## 🚀 Overview

This project presents an end-to-end **Retail Sales & Customer Feedback Analytics Dashboard** built using **Power BI**.
The dashboard analyzes a large dataset of **300K+ records**, transforming raw, messy data into meaningful business insights through data modeling, DAX, and interactive visualizations.

---

## 📂 Dataset

* Source: https://lnkd.in/gDf2ud6t
* Size: 300,000+ rows
* Data Type: Retail transactions, customer details, product information, shipping, and feedback

---

## 🎯 Objectives

* Analyze retail sales performance and trends
* Understand customer behavior and segmentation
* Evaluate product and brand performance
* Monitor shipping and operational efficiency
* Enable data-driven decision making through interactive dashboards

---

## 🔧 Tools & Technologies

* **Power BI** (Dashboard & Visualization)
* **Power Query** (Data Cleaning & Transformation)
* **DAX** (Measures & Calculations)
* **Data Modeling** (Star Schema Design)

---

## 🏗️ Data Modeling

The dataset was structured into a **Star Schema** to improve performance and scalability:

* **Fact Table:** Retail_Data
* **Dimension Tables:** Customer, Product, Shipping, Feedback, Date

✔ Established one-to-many relationships
✔ Created surrogate keys
✔ Standardized date, category, and product fields
